## 函数
函数就是解决问题的方法

函数的四个部分
 - 返回类型 : 函数的输出是什么东西。npy，有的函数是没有输出的
 - 函数名 : make_npy_happy，下划线分割，函数名要尽量起的见名知义
 - 形参列表 : 紧跟着函数名，小括号括起来的，函数的输入
 - 函数体 : 解决问题的过程，由一句或多句语句组成，语句末尾以分号结尾

```C++
npy make_npy_happy(angry_npy) {
    buy_bag;
    watch_tv_show;
    return npy;
}
```

main函数 ： 主函数
 - 程序是从main开始执行，到main结束
